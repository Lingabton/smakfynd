Så fungerar Smakfynd-poängen

Varje vin bedöms på tre saker: crowd-betyg (vad vanliga vindrickare tycker), expertrecensioner (kritiker som James Suckling, Decanter m.fl.) och prisvärde (pris jämfört med liknande viner). Hög kvalitet till lågt pris = hög poäng.
