Vilka uppgifter samlar vi?

Utan konto (alla besökare)

  • Sidvisningar — anonymt, via GoatCounter (cookieless analytics). Ingen personidentifiering.
  • Anonym användningsdata — vilka filter, sökningar och vinklick som görs. Lagras med ett slumpmässigt sessions-ID utan koppling till person.
  • Sparade viner — lagras lokalt i din webbläsare (localStorage). Vi har inte tillgång till dessa.

Med konto

  • E-postadress — för att identifiera ditt konto och synka viner mellan enheter.
  • Sparade viner — vilka viner du sparat och i vilka listor.
  • Nyhetsbrev-samtycke — om du valt att ta emot nyhetsbrev.

Varför samlar vi uppgifter?

  • För att ditt konto ska fungera (synka sparade viner)
  • För att skicka nyhetsbrev (endast om du samtyckt)
  • För att förbättra sajten (anonym statistik)

Rättslig grund

  • Samtycke — för nyhetsbrev och konto-skapande
  • Berättigat intresse — för anonym webbanalys

Hur länge sparar vi?

  • Konto-data — så länge kontot är aktivt. Raderas inom 30 dagar efter begäran.
  • Anonym analysdata — max 12 månader.
  • Nyhetsbrev-samtycke — tills du avregistrerar dig.

Var lagras data?

Data lagras hos Cloudflare (EU-region) och GitHub. Inga uppgifter överförs till tredje land utan adekvat skyddsnivå.

Dina rättigheter

Enligt GDPR har du rätt att:

  • Få tillgång till dina uppgifter
  • Rätta felaktiga uppgifter
  • Radera ditt konto och all data
  • Avregistrera dig från nyhetsbrev
  • Invända mot behandling
  • Klaga till Integritetsskyddsmyndigheten (IMY)

Kontakta oss på hej@smakfynd.se för att utöva dina rättigheter.

Cookies

Smakfynd använder inga cookies. Vi använder localStorage för att spara dina inställningar lokalt i din webbläsare, och sessionStorage för anonyma sessions-ID:n.

Tredjepartsleverantörer

  • Cloudflare — hosting, AI, databas (EU)
  • GitHub Pages — webbhosting
  • GoatCounter — cookieless analytics (EU)
  • Google Fonts — typsnitt (ingen persondata samlas)
