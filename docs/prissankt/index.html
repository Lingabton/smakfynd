Hur hittar vi prissänkningar?

Vi sparar priset på varje vin varje dag. När ett pris sjunker med minst 5% flaggar vi det här. Systembolaget har ingen offentlig lista över prissänkningar — men vi håller koll automatiskt. Listan uppdateras dagligen.
