# Målgruppsaudit — maj 2026

*Målgrupp: Vanliga vinköpare i Sverige som vill ha enkla, ärliga råd. Inte experter.*

---

## Sidor som talar deras språk

| Sida/feature | Varför det fungerar |
|---|---|
| Smakfynd-poängen (ett enda tal) | Svarar direkt på "är detta värt pengarna?" |
| Prissegmenten (under 100/150/200 kr) | Matchar hur folk tänker: "jag har X kr" |
| AI-vinmatcharen ("vad ska du äta?") | Tar bort expertkunskap, börjar med vardagen |
| FAQ "Är dyrare vin alltid bättre? Nej." | Bygger förtroende, avmystifierar |
| SB-länk direkt i varje vinkort | Pragmatiskt — leder till handling |
| Landningssidornas "topp 20"-format | Tar bort valet. "Vi har valt åt dig." |

## Sidor som talar FEL språk

### 1. Experttoner i copy (HÖG prioritet)

| Plats | Problem | Bättre |
|---|---|---|
| Överallt: "vindrickare" | Exkluderande — vanligt folk säger inte så | **"köpare"** eller **"användare"** |
| Card.jsx: "Jämförbara viner" | Formellt, oklart | **"Liknande viner"** eller **"Du kanske gillar"** |
| Card.jsx: "Kritikerfavorit" | Målgruppen bryr sig inte om kritiker | **"Högt betyg"** eller **"Populärt val"** |
| App.jsx: "kvalitet per krona" | Ekonomijargong | **"bästa vinet för pengarna"** |
| App.jsx: "Bästa köpet i sin kategori" | Vad är "sin kategori"? Oklart. | **"Bästa vinet för pengarna på Systembolaget"** |
| Metodsektionen: Bayesian shrinkage, datakällor | Ingen vanlig köpare läser detta | Förenkla till "Vi jämför betyg från tusentals köpare och experter" |

### 2. Landningssidor antar vinkunskap (HÖG prioritet)

| Slug | Problem | Varför det misslyckas |
|---|---|---|
| basta-malbec | "Malbec från Argentina är en favorit" | Förklarar inte vad Malbec smakar |
| basta-pinot-noir | "Elegant, fruktig, mångsidig. Från Bourgogne till Nya Zeeland" | Regioner som buzzwords utan kontext |
| basta-riesling | "Stentorra till sött" | "Stentorra" är jargong |
| basta-chardonnay | "Mineralisk till fyllig och fatlagrad" | "Mineralisk" och "fatlagrad" förklaras aldrig |
| basta-vin-fran-bordeaux | "Terroir är unikt... lagringspotential" | Expert-språk |
| vin-till-svamp | "Umami-djup, jordiga toner, tanninstarka" | Tre obegripliga ord i en mening |
| vin-for-nyborjare | "Druvor, regioner och uttryck" | Nybörjarsidan använder jargong! |

### 3. Smakprofil-sliderna (MEDEL prioritet)

Card.jsx visar tre skalor: Lätt–Fylligt, Stram–Fruktigt, Torrt–Sött med siffror 1–12. Vanliga köpare mappar inte vin till en 12-gradig skala. De tänker: "lätt eller tungt?", "sött eller inte?"

## Features som hjälper dem

| Feature | Bedömning |
|---|---|
| Smakfynd-poängen | Kärnan. Fungerar. |
| AI-vinmatchare | Perfekt koncept — men gömd bakom filter. Borde vara mer synlig. |
| Snabbkoll (butiksläge) | Exakt rätt — "jag står i butiken, vad säger du?" |
| Prisfilter | Ja — folk vet sin budget. |
| Typ-filter (rött/vitt/rosé/bubbel) | Ja — det enda de flesta behöver. |
| Spara favoriter | Bra för återbesökare, men "6 listor" är för mycket. |
| "Bästa vinet till [mat]"-sidor | Exakt rätt inramning. |

## Features som förvirrar dem

| Feature | Problem | Rekommendation |
|---|---|---|
| **Druv-filter** | Vanlig köpare vet inte vad Tempranillo är | Dölj bakom "Fler filter" |
| **Region-filter** | Priorat, Languedoc, Rhône — ingenting för målgruppen | Dölj bakom "Fler filter" |
| **6 sorteringsalternativ** | "Expert", "Crowd", "Pris ↑", "Pris ↓", "Prissänkt" — för många val | Behåll 2: "Bäst för pengarna" + "Lägst pris" |
| **Smak-skalor (kropp/frukt/sötma)** | 12-gradig skala är expert-UX | Förenkla till ord: "Lätt", "Medel", "Fylligt" |
| **Källarfunktion + provningsanteckning** | Vinsamlarfeature. Målgruppen dricker vinet ikväll. | Ta bort eller göm djupt |
| **Kritiker-badges** (Suckling, Decanter, Falstaff) | Ingen vet vilka dessa är | Visa bara "Expertbetyg: 7.2/10" — en rad |
| **"Data: Komplett/Partiell/Begränsad"** | Transparens som skapar osäkerhet | Ta bort — eller visa bara vid låg data |
| **Score-uppdelning (Kval % + Pris %)** | Två staplar fragmenterar budskapet | Visa bara poängen + en etikett ("Bra köp") |

---

## Sammanfattning: tre strategiska skift

1. **Språket ska vara vardagligt.** Aldrig "vindrickare", "struktur", "terroir", "mineralisk". Alltid "köpare", "smak", "det här vinet smakar körsbär och krydda."

2. **Göm expertfunktioner.** Druv/region-filter, källare, provningsnoter, kritiker-badges — bakom en "Mer"-knapp eller borttagna helt.

3. **Promota AI-matcharen.** Den är den mest målgruppsriktiga funktionen men den är gömd. Den borde vara lika synlig som sökfältet.
